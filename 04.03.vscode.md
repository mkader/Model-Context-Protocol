#### Consuming a server from GitHub Copilot Agent mode
* VSC and GitHub Copilot can act as a client and consume an MCP Server.

#### Usage

You can control your MCP server in two different ways:

- User interface, you will see how this is done later in this chapter.
- Terminal, it's possible to control things from the terminal using the `code` exectuable:

  To add an MCP server to your user profile

  ```
  code --add-mcp "{\"name\":\"my-server\",\"command\": \"uvx\",\"args\": [\"mcp-server-fetch\"]}"
  ```

#### Approach

Here's how we need to approach this at high level:

- Configure a file to find our MCP Server.
- Start up/Connect to said server to have it list its capabilities.
- Use said capabilities through GitHub Copilot Chat interface.

#### Exercise: Consuming a server
* configure VSC to find your MCP server so that it can be used from GitHub Copilot Chat interface.

* 1.Prestep, enable MCP Server discovery
    * Go to `File -> Preferences -> Settings` in VSC
    * Search for "MCP" and enable `chat.mcp.discovery.enabled` in the settings.json file.

* 2.Create config file(.vscode/MCP.json) in your project root, place i ( folder)
    ```json
    {
        "inputs": [],
        "servers": {
           "hello-mcp": {
               "command": "node",
               "args": [
                   "build/index.js"
               ]
           }
        }
    }
    ```
    * start a server written in Node.js using `command` and `args`.

  * 3.Start the server
      * Locate your entry in *mcp.json* and make sure you find the "play" icon:
          ![Starting server in Visual Studio Code](./assets/vscode-start-server.png)  
      * Click the "play" icon, you should see the tools icon in the GitHub Copilot Chat increase the number of available tools.
      * If you click said tools icon, you will see a list of registered tools.
      * You can check/uncheck each tool depending if you want GitHub Copilot to use them as context:
          ![Starting server in Visual Studio Code](./assets/vscode-tool.png)
      * To run a tool, type a prompt that you know will match the description of one of your tools, for example a prompt like so "add 22 to 1":
          ![Running a tool from GitHub Copilot](./assets/vscode-agent.png)
